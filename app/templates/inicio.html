Métodos de Optimización

Resuelve problemas de programación lineal utilizando diferentes algoritmos

Método Simplex

Algoritmo clásico para resolver problemas de programación lineal estándar. Ideal para problemas con restricciones de desigualdad.

Resolver
Método Gran M

Utiliza variables artificiales con penalización para resolver problemas con restricciones de igualdad y desigualdad.

Resolver
Método Dos Fases

Resuelve el problema en dos etapas: primero encuentra una solución factible, luego optimiza la función objetivo.

¿Cómo usar esta aplicación?
Formato de Entrada
  • Función Objetivo (c): Separar coeficientes con comas. Ej: 3,2,1
  • Matriz A: Una fila por línea, coeficientes separados por comas
  • Vector b: Separar valores con comas. Ej: 12,8,6
Consejos
  • Marca "Mostrar iteraciones" para ver el proceso paso a paso
  • Para Gran M y Dos Fases, especifica restricciones de igualdad
  • Los índices de restricciones empiezan desde 0
