🔺 Métodos de Optimización

API para resolver problemas de programación lineal

Bienvenido a la API de Métodos de Optimización

Esta API permite resolver problemas de programación lineal utilizando diferentes métodos de optimización.

Disponemos de las siguientes implementaciones:

  • Método Simplex: Para problemas de programación lineal estándar.
  • Método de la Gran M: Para problemas con restricciones de igualdad o variables artificiales.
  • Método de las Dos Fases: Alternativa al método de la Gran M para problemas con restricciones de igualdad.

Documentación de la API

GET /api/casos

Obtiene la lista de todos los ejemplos guardados.

Respuesta: Array de objetos JSON con los casos de ejemplo.

POST /api/resolver/simplex

Resuelve un problema utilizando el método Simplex.

Cuerpo de la petición:

{
  "c": [5, 7],
  "A": [[2, 3], [1, 1], [1, 0]],
  "b": [12, 5, 3],
  "minimize": false,
  "track_iterations": true
}

Respuesta: Objeto JSON con la solución, valor óptimo y opcionalmente el historial de iteraciones.

POST /api/resolver/granm

Resuelve un problema utilizando el método de la Gran M.

Cuerpo de la petición:

{
  "c": [5, 7],
  "A": [[2, 3], [1, 1], [1, 0]],
  "b": [12, 5, 3],
  "minimize": false,
  "eq_constraints": [0],
  "track_iterations": true
}

Respuesta: Objeto JSON con la solución, valor óptimo y opcionalmente el historial de iteraciones.

POST /api/resolver/dosfases

Resuelve un problema utilizando el método de las Dos Fases.

Cuerpo de la petición:

{

  "b": [12, 5, 3],
  "minimize": false,
  "eq_constraints": [0],
  "track_iterations": true

Respuesta: Objeto JSON con la solución, valor óptimo y opcionalmente el historial de iteraciones.
